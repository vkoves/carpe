handlePosition(elem, ui)

Adjusts position of a schedule item when user is done dragging it, so that the event card is in a valid place on the column (i.e., not past the top or bottom bounds of the column), and snaps the event to the hourly grid on the schedule. This function is called on a draggable object's stop() event, which is when the user "lets go" of the draggable object.

elem string The jQuery selector for an event on user's schedule
